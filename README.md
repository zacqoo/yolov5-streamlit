# yolov5-streamlit

Deploy [YOLOv5](https://github.com/ultralytics/yolov5/releases/tag/v5.0) detection with [Streamlit](https://github.com/streamlit/streamlit)

# Installation

```
pip install -r requirements.txt
```

# Run

```
streamlit run main.py
```
